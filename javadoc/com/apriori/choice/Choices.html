Class Choices

  • java.lang.Object
    • com.apriori.choice.Choices

  • public class Choices
extends Object
    Utility methods and interfaces for working with Choices. The basic Choice interface (and its enclosed interfaces, like Choice.Ops3, etc.) contain abstract operations for working with a choice out of N or more options. Additional interfaces are included herein with more concrete choices. For example, Choice.Ops3 contains operations for choices that have three or more options; whereas the enclosed Choices.Choices3 extends those operations for choices that have exactly three options.
