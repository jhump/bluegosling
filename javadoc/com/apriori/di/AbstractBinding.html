Class AbstractBinding

  • All Implemented Interfaces:
    Binding

    @UsingScope(scope=javax.inject.Singleton.class,
            impl=SingletonScoper.class)
public abstract class AbstractBinding
extends Object
implements Binding
    An abstract base class for bindings. Sub-classes need only implement defineBindings() and therein use the various bind(com.apriori.di.Key<T>) methods to define bindings. The handling of the Binding interface methods is automatically handled by this class.
