Class ListenableFutureTask<T>

    Direct Known Subclasses:

    public class ListenableFutureTask<T>
extends FutureTask<T>
implements ListenableFuture<T>
    A future task that implements the ListenableFuture interface. This builds on the FutureTask implementation provided by the Java runtime library and extends it with additional API.

    Another implementation of the ListenableFuture interface that doesn't necessarily represent a runnable task is SimpleListenableFuture. That other class also implements some of the new API more efficiently since it doesn't operate within constraints inherited from FutureTask (which was designed to provide a more simplistic API).
