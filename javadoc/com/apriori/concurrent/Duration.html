com.apriori.concurrent

Class Duration

  • java.lang.Object
    • com.apriori.concurrent.Duration
  • All Implemented Interfaces:
    Comparable<Duration>

    public class Duration
extends Object
implements Comparable<Duration>
    Represents a duration of time that is backed by an integral length and a unit.

    When two durations are added or subtracted, the result will convert to the finest common unit that won't result in overflow or underflow. If a result unit is TimeUnit.DAYS but still overflows or underflows, the result will be clipped to Long.MAX_VALUE or Long.MIN_VALUE, which mirrors the behavior of converting between units using TimeUnit.convert(long, TimeUnit).

    This provides a sane way to compare and use durations of varying units. To provide the duration to APIs that accept a long and a TimeUnit, simply pass duration.length() and duration.unit().

    Note: this class has a natural ordering that is inconsistent with equals.

      • plus

        public Duration plus(long other,
            TimeUnit otherUnit)
        Returns the sum of this duration and the specified duration. The unit of the returned sum will be the finer of this or the specified duration's unit() unless that would result in overflow or underflow. In such case, a coarser unit is chosen, the finest unit that can represent the sum without overflow or underflow.

        If the result would be greater than Long.MAX_VALUE days then the result is clipped to Long.MAX_VALUE. Similarly, if the result would be less than Long.MIN_VALUE days, the result is clipped to Long.MIN_VALUE.

        Parameters:
        other - the other addend
        otherUnit - the unit of the other addend
        Returns:
        the sum of this duration plus the one specified
