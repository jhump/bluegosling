      • assemble

        public static <T> FutureListener<T> assemble(Sink<? super T> onSuccess,
                             Sink<? super Throwable> onFailure,
                             Runnable onCancel)
        Assembles a listener using all of the specified callbacks. Each callback is invoked depending on the disposition of the completed futures.

        This method uses only functional interfaces, which will make it easier to assemble listeners in Java 8.

        Parameters:
        onSuccess - invoked when the completed future is successful; the future's result is passed to the sink
        onFailure - invoked when the completed future has failed; the cause of failure is passed to the sink
        onCancel - invoked when the completed future is cancelled
        Returns:
        a listener that will call one of the specified callbacks when invoked
