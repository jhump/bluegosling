com.apriori.concurrent

Interface ScheduledTask<V,T>

  • Type Parameters:
    V - the type of value returned upon completion of the task
    T - the type of the actual task: Callable, Runnable, or RunnableWithResult
    All Superinterfaces:
    Awaitable, Cancellable, Comparable<Delayed>, Delayed, Future<V>, ListenableFuture<V>, ListenableScheduledFuture<V>, ScheduledFuture<V>
    All Known Subinterfaces:
    CallableScheduledTask<V>, RunnableScheduledTask

    public interface ScheduledTask<V,T>
extends ListenableScheduledFuture<V>
    The type of ScheduledFuture returned by submissions to a ScheduledTaskManager. This extension provides various utility methods as well as access to the ScheduledTaskDefinition for which the task was created.

    A ScheduledTask basically represents a single invocation of a ScheduledTaskDefinition.
