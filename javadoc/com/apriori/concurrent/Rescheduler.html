Interface Rescheduler<T>

  • Type Parameters:
    T - the type of future value for which a subsequent calculation may be rescheduled

    public interface Rescheduler<T>
    Handles rescheduling a periodic task. This computes the next scheduled execution time for a repeating task. An interface provides greater flexibility than does simple API where the only options for repeating tasks are to run at a fixed rate or with a fixed delay. Complicated recurring schedules can be implemented this way.
