Interface Atom.Watcher<T>

  • Type Parameters:

    Enclosing interface:
    Atom<T>

    public static interface Atom.Watcher<T>
    Watches an atom for changes to its value. Implementations of this interface must be thread-safe since they may be called concurrently from multiple threads.
