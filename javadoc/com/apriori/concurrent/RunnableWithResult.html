Interface RunnableWithResult<V>

  • Type Parameters:
    V - the type of the result

    public interface RunnableWithResult<V>
    A Runnable combined with a specified result value. This interface represents the pair of parameters as used by ExecutorService.submit(Runnable, V) and Executors.callable(Runnable, V).

    None of the APIs in this package actually use this interface. It is just used for type-safety to parameterize TaskDefinitions.
