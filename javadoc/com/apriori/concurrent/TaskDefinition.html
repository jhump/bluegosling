com.apriori.concurrent

Interface TaskDefinition<V,T>

  • Type Parameters:
    V - the type of value returned upon completion of the task
    T - the type of the actual task: Callable, Runnable, or RunnableWithResult
    All Known Subinterfaces:
    ScheduledTaskDefinition<V,T>

    public interface TaskDefinition<V,T>
    The definition for a task that will be executed by a BetterExecutorService. This includes several configuration options that are not possible using a normal ScheduledExecutorService.

      • isRepeating

        boolean isRepeating()
        Returns whether or not this is a repeating task vs. a one-time task. A task is a one-time task if its ScheduleNextTaskPolicy is ScheduleNextTaskPolicies.NEVER. Otherwise, it is considered a repeating task.
        Returns:
        true if this is a repeating task; false otherwise

        boolean isFixedRate()

        Returns:

      • periodDelayMillis

        long periodDelayMillis()
        Returns the period (for fixed rate tasks) or delay (for fixed delay tasks) for this task. If this is not a repeating task, zero is returned. If this is a repeating task and this method returns zero, then subsequent invocations are scheduled immediately upon completion of the previous invocation, running the tasks sequentially as quickly as possible.
        Returns:
        the period or delay, in milliseconds, for this task

      • scheduleNextTaskPolicy

        ScheduleNextTaskPolicy<? super V,? super T> scheduleNextTaskPolicy()
        Returns the ScheduleNextTaskPolicy for this repeating task. If this task is not a repeating task, this returns ScheduleNextTaskPolicies.NEVER.
        Returns:
        the ScheduleNextTaskPolicy for this task

      • maxHistorySize

        int maxHistorySize()
        Returns the maximum size of invocation history to maintain for this task. This is the maximum size of the collection returned by ScheduledTaskDefinition.history() upon scheduling this task definition. If the task repeats more times than this size, information for older invocations is dropped.
        Returns:
        the maximum size of invocation history

      • listeners

        Set<ScheduledTask.Listener<? super V,? super T>> listeners()
        Returns the set of listeners for this task. These objects are notified upon completion (including failures/abnormal completions) of each invocation of the task.

        The set of listeners can actually be changed after the task definition has been scheduled using ScheduledTaskDefinition.addListener(ScheduledTask.Listener) and ScheduledTaskDefinition.removeListener(ScheduledTask.Listener).

        Returns:
        the set of listeners for this task

      • uncaughtExceptionHandler

        Thread.UncaughtExceptionHandler uncaughtExceptionHandler()
        Returns the Thread.UncaughtExceptionHandler for this task. If any invocation throws an uncaught exception, this handler will be dispatched to handle it. If the tasks's handler is never assigned (i.e. null) then the thread's default handler is used.
        Returns:
        the UncaughtExceptionHandler for this task
