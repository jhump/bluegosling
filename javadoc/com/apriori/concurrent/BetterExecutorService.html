com.apriori.concurrent

Class BetterExecutorService

  • java.lang.Object
    • com.apriori.concurrent.BetterExecutorService
  • All Implemented Interfaces:
    Executor, ExecutorService, ScheduledExecutorService

    public class BetterExecutorService
extends Object
implements ScheduledExecutorService
    An extension and enhancement to the API provided by ScheduledExecutorService. This implementation provides greater configurability over scheduled tasks and greater optics into the results of executions of the tasks.
