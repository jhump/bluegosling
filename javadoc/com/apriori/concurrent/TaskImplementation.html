Class TaskImplementation<V>

  • java.lang.Object
    • com.apriori.concurrent.TaskImplementation<V>
  • Type Parameters:
    V - the type of result produced by the task
    All Implemented Interfaces:
    Callable<V>

    public abstract class TaskImplementation<V>
extends Object
implements Callable<V>
    The implementation for a task that is executed using a ScheduledTaskManager. The task can be implemented by either a Callable or a Runnable.

    This class implements Callable. Calling this task will forward to the underlying implementation, so this object also serves as an adapter from any type of implementation to Callable.
