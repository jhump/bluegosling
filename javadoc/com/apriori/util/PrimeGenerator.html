Class PrimeGenerator<T extends Number & Comparable<T>>

  • java.lang.Object
    • com.apriori.util.PrimeGenerator<T>
  • Type Parameters:
    T - the type of numeric value generated
    All Implemented Interfaces:
    Iterator<T>

    public abstract class PrimeGenerator<T extends Number & Comparable<T>>
extends Object
implements Iterator<T>
    Generates a sequence of prime numbers. The sequence enumerates all primes, starting with two. The sequence is complete when the maximum possible range is exhausted. For example, a generator of prime bytes stops after the largest prime that is less than or equal to 127 (which is the maximum value that can be represented by a byte).

    This is implemented using a Sieve of Eratosthenese. This implementation is an incremental sieve, so it has no fixed upper bound other than being bound by heap space. Generating the nth prime requires O(n) memory, so a lot of memory is required for very large n.

    The phrase "current numeric type" in the documentation refers to the concrete type for type argument T. For example, with a PrimeGenerator<Long>, the "current numeric type" would be long.
