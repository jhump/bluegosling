      • chain

        public static <I,O> PartialFunction<I,O> chain(PartialFunction<? super I,? extends O> function1,
                               PartialFunction<? super I,? extends O> function2)
        Returns a partial function that chains two partial functions together. If the first function does not support an input, then the second function will be invoked. The result for a given input will be absent only if neither function can compute it. If both functions can compute a result for an input, the result from the first function will be used.
        Parameters:
        function1 - the first partial function
        function2 - the second partial function
        Returns:
        a partial function that invokes the first function and then, if necessary, the second in order to compute result

      • lift

        public static <I,O> Function<I,O> lift(PartialFunction<? super I,? extends O> function,
                       Class<? extends RuntimeException> throwIfUndefined)
        Lifts a partial function to a regular function that throws a custom exception for incomputable inputs. The returned value of the regular function is not optional.

        This form of the method does not provide a real fallback strategy for inputs that cannot be computed. Instead, the returned function will throw a custom exception when given an input that isn't supported. The specified exception type must be unchecked (since a partial function cannot throw a checked exception due to its signature). It also must have a default, no-argument constructor which will be used to construct instances of the exception.

        Parameters:
        function - a partial function
        throwIfUndefined - the type of exception to
        Returns:
        a regular function that throws an exception for unsupported inputs
