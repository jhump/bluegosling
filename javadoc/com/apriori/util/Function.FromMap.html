      • Function.FromMap

        public Function.FromMap(Map<I,O> map)
        Constructs a new function that performs look-ups against the specified map. When the function is invoked for a given argument, the map is inspected. If the map contains a key equal to the specified argument, then the corresponding value will be the function's result. If there is no such key then null is returned.
        Parameters:
        map - the map that backs this function
        See Also:
        strict()
