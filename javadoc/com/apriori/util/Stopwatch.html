com.apriori.util

Class Stopwatch

  • java.lang.Object
    • com.apriori.util.Stopwatch

  • public class Stopwatch
extends Object
    A utility for measuring elapsed time. It can be stopped and restarted and the resulting elapsed time will represent time elapsed during periods when the stopwatch was "running". It also has a "lap" function so that data can be collected and retained for multiple iterations or laps.

    This class is threadsafe, so a single instance could be shared by multiple threads if need be.

    • Method Detail

      • start

        public void start()
        Starts the stopwatch. Once started, it is "running". Time that elapses while the clock is running can be queried via read().

        If the stopwatch is already running, this method does nothing.

      • stop

        public void stop()
        Stops the stopwatch. Once stopped, it is no longer "running". Time that elapses while the watch is stopped will not be measured and will not impact queries.
        See Also:
        reset()

      • lap

        public void lap()
        Records the stopwatch's elapsed time as a lap measurement and resets the elapsed time. If the stopwatch is stopped when this method is called, it will still be stopped when this method returns.

      • read

        public long read()
        Queries for the elapsed time measured by the stopwatch, in milliseconds. This returns the total elapsed time during which the stopwatch was "running" since the last call to either lap() or reset().
