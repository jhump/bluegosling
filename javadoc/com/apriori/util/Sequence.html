Interface Sequence<T,X extends Throwable>

  • Type Parameters:
    T - the type of elements in the sequence
    X - the type of exception that may be thrown during sequence generation
    All Known Subinterfaces:
    UncheckedSequence<T>

    public interface Sequence<T,X extends Throwable>
    Represents a sequence of items produced by a Generator. This is very similar to the Iterator interface, but it allows for propagation of exceptions that the generator may throw.

      • next

        T next()
       throws X extends Throwable
        Returns the next element in the sequence. This transfers control to the generator and returns when the generator provides the next value. If the generator terminates, either in failure or successfully, then an exception is thrown.
        Returns:
        the next element in the sequence
        Throws:
        X - if the generator fails and throws an exception
        SequenceFinishedException - if the generator completes successfully, indicating that there are no more elements in the sequence

      • asIterator

        Iterator<T> asIterator()
        Returns a view of this sequence as an Iterator. This may transfer control to the generator thread during calls to hasNext() to determine if there is another item by letting the generator actually produce it. As such, both next() and hasNext() methods may throw an (unchecked) exception if the generator fails.
        Returns:
        a view of this sequence as an Iterator
