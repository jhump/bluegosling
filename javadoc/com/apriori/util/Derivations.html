Class Derivations

  • java.lang.Object
    • com.apriori.util.Derivations

  • public final class Derivations
extends Object
    Utility methods for determining if types are derived from one another. A type, TypeA, is derived from another type, TypeB, if any of the following are true:
    • TypeA implements DerivedFrom<TypeB>
    • TypeA implements DerivedFrom<TypeC> and TypeC is derived from TypeB
    • TypeA is annotated with @IsDerivedFrom(TypeB.class)
    • TypeA is annotated with @IsDerivedFrom(TypeC.class) and TypeC is derived from TypeB

      • getDirectDerivationSources

        public static Set<Class<?>> getDirectDerivationSources(Class<?> derivedType)
        Gets the sources from which the specified type is directly derived. A type, TypeA, is directly derived from another type, TypeB, if it implements DerivedFrom<TypeB> or if it is annotated with @IsDerivedFrom(TypeB.class).

        The returned set will have between zero and two entries. It could have two entries if the specified type both implements DerivedFrom and is annotated with @IsDerivedFrom and the two referenced sources differ. If the class neither implements that interface nor is annotated then the returned set will be empty.

        Parameters:
        derivedType - the type to check
        Returns:
        the set of types from which the specified type is directly derived

      • getAllDerivationSources

        public static Set<Class<?>> getAllDerivationSources(Class<?> derivedType)
        Gets all sources, both direct and indirect, from which the specified type is derived. This will return all types for which isDerivedFrom(Class, Class) would return true. Since a class is considered "derived from" itself, then the returned set also includes the specified type.

        This includes any types from which the specified type is directly derived. It also includes all types from which those direct sources are derived, etc.

        Parameters:
        derivedType - the type to check
        Returns:
        the set of all types from which the specified type is derived, directly and indirectly
