Interface PartialFunction<I,O>

  • Type Parameters:

    All Superinterfaces:
    Function<I,Optional<O>>

    public interface PartialFunction<I,O>
extends Function<I,Optional<O>>
    A function that does not fully map the input domain to the output domain. This is reflected in the API by returning no value when a result cannot be computed for a given input.

    Note that the use of Optional means that null results for a valid input are not allowed.
