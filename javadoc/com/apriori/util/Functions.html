        public static <I,O> PartialFunction<I,O> unlift(Function<? super I,? extends O> function)
        Returns a partial function for the specified function. In spirit, this is the opposite of "lifting" a partial function to a total function. Since partial functions do not support null results, the returned partial function will be undefined for any inputs where the specified function returns null. All other values in the input domain behave as expected and produce a result.
        Parameters:
        function - a function
        Returns:
        a partial function that is defined for all values in the input domain except the ones where the specified function returns null
