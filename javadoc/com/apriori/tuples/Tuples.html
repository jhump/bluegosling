      • fromArray

        public static Tuple fromArray(Object... array)
        Creates a new tuple with the same items as the specified array. The first item in the tuple will be the first item in the array, and so on. If the specified array is empty, Empty.INSTANCE will be returned. If the array has exactly one, two, three, four, or five elements then a Unit, Pair, Trio, Quartet, or Quintet will be returned, respectively. If the array has more than five elements than an NTuple is returned.
        Parameters:
        array - the array
        Returns:
        a tuple with the same items as the array
