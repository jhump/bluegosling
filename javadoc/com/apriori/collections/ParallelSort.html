com.apriori.collections

Class ParallelSort

  • java.lang.Object
    • com.apriori.collections.ParallelSort

  • public class ParallelSort
extends Object
    A utility for sorting large lists using multiple CPUs/cores. This is a simplistic parallel sort that breaks the incoming list into chunks, sorts all of the chunks in parallel, and then finally merges the chunks. Only the sorting of chunks is parallelized.
