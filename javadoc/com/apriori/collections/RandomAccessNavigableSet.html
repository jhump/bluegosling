Interface RandomAccessNavigableSet<E>

  • Type Parameters:
    E - the type of element in the set
    All Superinterfaces:
    Collection<E>, Iterable<E>, NavigableSet<E>, RandomAccessSet<E>, Set<E>, SortedSet<E>

    public interface RandomAccessNavigableSet<E>
extends NavigableSet<E>, RandomAccessSet<E>
    A navigable set that also provides random access to its members. The elements in the set are ordered according to the set's Comparator or (if there is none) according to the elements' natural ordering.
