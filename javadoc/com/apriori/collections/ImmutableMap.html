    public interface ImmutableMap<K,V>
extends Iterable<ImmutableMap.Entry<K,V>>
    An immutable, read-only set of key+value mappings. This interface is similar to the standard Map interface except that it defines no mutation operations.

    The remove() operation of an immutable map's Iterable.iterator() always throws UnsupportedOperationException. Similarly the Iterable.iterator() for any of the collection views of the map will also always throw UnsupportedOperationException.
