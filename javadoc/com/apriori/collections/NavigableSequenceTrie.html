      • prefixMap

        NavigableSequenceTrie<K,V> prefixMap(List<K> prefix)
        Description copied from interface: SequenceTrie
        Generates a view of all mappings whose keys have the specified prefix. The returned map's keys will just be the tail sequences: the specified sequence, which prefixes all of its keys, will be omitted.

        This is effectively shorthand for repeated calls to SequenceTrie.prefixMap(Object), like so:

         SequenceTrie result = someTrie;
 for (K value : prefix) {
   result = result.prefixMap(value);
 }
        Specified by:
        prefixMap in interface SequenceTrie<K,V>
        Parameters:
        prefix - a key prefix
        Returns:
