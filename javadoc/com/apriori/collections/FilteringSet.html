    public class FilteringSet<E>
extends FilteringCollection<E>
implements Set<E>
    A filtered view of another set. This is a wrapper that elides all members of the wrapped set that do not match a given predicate. Changes to the underlying set will be visible through the filtering set (or at least those changes that effect elements that match the predicate).

    Attempts to add items to the filtering set that do not match the predicate will throw IllegalArgumentExceptions.
