    public class FilteringIterable<E>
extends Object
implements Iterable<E>
    A filtered view of another iterable. This is a wrapper that elides all members of the wrapped iterable that do not match a given predicate. Changes to the underlying iterable will be visible through the filtering iterable (or at least those changes that effect elements that match the predicate).
