    public class TransformingIterable<I,O>
extends Object
implements Iterable<O>
    An iterable whose elements are the results of applying a function to another iterable. This iterable is simply a wrapper. Changes to the underlying iterable are visible in the transformed iterable. Accessing elements incurs calls to the transforming function.

    Functions that perform the transformations should be deterministic so that a stable, unchanging iterable does not appear to be mutating when accessed through this transforming wrapper.
