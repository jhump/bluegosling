    public class FilteringList<E>
extends FilteringCollection<E>
implements List<E>
    A filtered view of another list. This is a wrapper that elides all members of the wrapped list that do not match a given predicate. Changes to the underlying list will be visible through the filtering collection (or at least those changes that effect elements that match the predicate).

    Attempts to add/set items in the filtering list that do not match the predicate will throw IllegalArgumentExceptions.
