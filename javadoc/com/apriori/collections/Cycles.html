      • asListIterator

        public static <E> ListIterator<E> asListIterator(Cycle<E> cycle)
        Returns a view of the specified cycle as a ListIterator. Invoking ListIterator.next() is equivalent to calling Cycle.current() followed by Cycle.advance(). Invoking ListIterator.previous() is equivalent to calling Cycle.retreat() followed by Cycle.current().

        The list indices returned by this iterator do not have much meaning. The initial element is considered to be list index zero. However, little attempt is made to keep this consistent, especially in the face of concurrent modifications being made to the sequence through the standard Cycle interface, or through other ListIterator views. Specifically, the first call to ListIterator.next() will be the item at list index zero. Each call to ListIterator.next() increments the current list index. Each call to ListIterator.previous() decrements it. If the index would be become negative, it is set to size() - 1. If it would become greater than or equal to size(), it is set to zero. Unless the sequence is empty, ListIterator.hasNext() and ListIterator.hasPrevious() always return true, and movement through the sequence is cyclic and unending.

        Returns:
        a view of this cycle as a list iterator
