      • size

        public int size()

        This implementation actually counts the number of entries within the sub-map's bounds using the AbstractRandomAccessNavigableMap.entrySet()'s iterator. This runs in linear time, O(n), but the result is memo-ized between calls. It will only count the number of entries again if it detects that the main map's AbstractNavigableMap.modCount has changed since the last time the entries were counted.

      • getModCount

        protected int getModCount()
        Returns the main map's modification count. This allows iterators over sub-map views to properly fail-fast.

        Since a AbstractRandomAccessNavigableMap.RandomAccessSubMap extends AbstractRandomAccessNavigableMap, it also inherits a AbstractNavigableMap.modCount field. For sub-maps, that field is compared to the main maps' value to detect when the sub-map's memo-ized size and index bounds are stale and elements need to be counted.
