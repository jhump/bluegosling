    public interface MeldableOrderedQueue<E,Q extends MeldableOrderedQueue<? extends E,?>>
extends OrderedQueue<E>
    A OrderedQueue that can be merged with another efficiently. Several types of priority queue implementations support efficient merge operations (such as binomial, skew, and fibonacci heaps). This category of priority queue is often known as a meldable queue.

      • mergeFrom

        boolean mergeFrom(Q other)
        Merges the specified queue into this queue. The other queue will be empty when this operation completes and this queue will have the union of its own prior contents and those of the specified queue.

        This is effectively the same as the following:

         MeldablePriorityQueue<T> queue1, queue2;
 // merge queue2 into queue1
 queue1.addAll(queue2);
 queue2.clear();
        The main difference is that this specialized method can perform the merge in constant time (possibly amortized constant time). Without this method, such an operation would likely instead take linear or O(n log n) time.
        Parameters:
        other - the queue whose contents are merged into this queue
        Returns:
        true if both queues were modified as a result; false if the specified queue was empty
