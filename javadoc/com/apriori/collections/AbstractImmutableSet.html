    public abstract class AbstractImmutableSet<E>
extends AbstractImmutableCollection<E>
implements ImmutableSet<E>
    An abstract base class for implementing an ImmutableSet. This provides implementations for all methods except ImmutableCollection.size() and Iterable.iterator().

    Nothing in this base class enforces set invariants. The only difference between this class and its super-class, AbstractImmutableCollection, is that this one implements ImmutableSet.equals(Object) and ImmutableSet.hashCode().

    Sub-classes will likely override AbstractImmutableCollection.contains(Object) to provide better performance. The default implementation (inherited from AbstractImmutableCollection) performs a linear scan through all elements in the set.

        Returns the hash code value for this set. The hash code of a set is defined to be the sum of the hash codes of the elements in the set, where the hash code of a null element is defined to be zero.
