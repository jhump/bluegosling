      • prefixMap

        SequenceTrie<K,V> prefixMap(List<K> prefix)
        Generates a view of all mappings whose keys have the specified prefix. The returned map's keys will just be the tail sequences: the specified sequence, which prefixes all of its keys, will be omitted.

        This is effectively shorthand for repeated calls to prefixMap(Object), like so:

         SequenceTrie result = someTrie;
 for (K value : prefix) {
   result = result.prefixMap(value);
 }
        Parameters:
        prefix - a key prefix
        Returns:
        a view of this map that represents the subset of keys with the specified prefix

      • prefixMap

        SequenceTrie<K,V> prefixMap(List<K> prefix,
                          int numComponents)
        Generates a view of all mappings whose keys have the specified prefix. A maximum number of components of the specified prefix are considered. So the effective prefix is the specified sequence but truncated to no more than the specified number of elements.
        Parameters:
        prefix - a key prefix
        numComponents - the maximum number of components of the prefix
        Returns:
        a view of this map that represents the subset of keys with the specified prefix
