    public interface ImmutableCollection<E>
extends Iterable<E>
    An immutable, read-only collection. This interface is similar to the standard Collection interface except that it defines no mutation operations.

    The remove() operation of an immutable collection's Iterable.iterator() always throws UnsupportedOperationException.
