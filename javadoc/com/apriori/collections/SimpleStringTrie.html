  • java.lang.Object

    All Implemented Interfaces:

    public abstract class SimpleStringTrie<V>
extends Object
implements NavigableCompositeTrie<CharSequence,Character,V>
    A trie that has String keys. This is nearly the same as using a SortedArrayCompositeTrie but has a couple of useful advantages:
    • This trie supports proper localized sorting of keys via a Collator.
    • This trie is a little more efficient since it uses arrays of primitives (char) instead of the boxed counterpart (Character).
