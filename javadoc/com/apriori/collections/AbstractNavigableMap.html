Class AbstractNavigableMap<K,V>

    • com.apriori.collections.AbstractNavigableMap<K,V>
  • Type Parameters:
    K - the type of keys in the map

    All Implemented Interfaces:
    Map<K,V>, NavigableMap<K,V>, SortedMap<K,V>
    Direct Known Subclasses:

    public abstract class AbstractNavigableMap<K,V>
extends Object
implements NavigableMap<K,V>
    An abstract base class for NavigableMap implementations. Concrete classes only need to implement methods that perform search operations. This abstract class then implements everything else, including keySet() and entrySet(), in terms of those operations. Mutable maps must also implement store and remove operations.

    This class is not thread-safe. So most of the method implementations herein cannot be used to correctly implement a ConcurrentNavigableMap.

    Because the NavigableMap interface is so much broader (and thus different) than the plain Map interface, this class and its usage bear little resemblance to the JRE's AbstractMap base class. As such, this class does not extend AbstractMap.
