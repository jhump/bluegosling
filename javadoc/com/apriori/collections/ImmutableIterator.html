Interface ImmutableIterator<E>

  • Type Parameters:
    E - the type of elements returned from this iterator
    All Known Subinterfaces:
    ImmutableIterator.Bidi<E>

    public interface ImmutableIterator<E>
    A read-only iterator. It is named Immutable for consistency with other immutable collection interfaces. However, an iterator cannot be truly immutable since its "current location" is state that must be mutated as it moves through the sequence. "Immutable" in this case instead refers to the underlying sequence of values. The main difference between this class and its normal Iterator kin is that this interface does not support any mutations to the stream (e.g. no Iterator.remove() method).
