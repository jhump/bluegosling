    All Implemented Interfaces:
    Iterable<E>, Collection<E>, BlockingDeque<E>, BlockingQueue<E>, Deque<E>, List<E>, Queue<E>, RandomAccess

    public class RandomAccessArrayBlockingDeque<E>
extends RandomAccessArrayDeque<E>
implements BlockingDeque<E>
    A BlockingDeque implementation based on a RandomAccessArrayDeque. Like its super-class, it is based on a growable array and exposes random access operations on its contents.
