Interface BidiListIterator<T>

  • Type Parameters:
    T - the type of element in the list
    All Superinterfaces:
    BidiIterator<T>, Iterator<T>, ListIterator<T>

    public interface BidiListIterator<T>
extends BidiIterator<T>, ListIterator<T>
    A list iterator that also implements BidiIterator. List iterators already support iteration in both directions. Implementing this interface only adds one new operation: reverse().
