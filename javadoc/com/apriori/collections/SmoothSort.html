Class SmoothSort

  • java.lang.Object
    • com.apriori.collections.SmoothSort

  • public final class SmoothSort
extends Object
    Sorts arrays and collections using the Smoothsort algorithm. This algorithm is very similar to heap sort, but it uses a Leonardo heap instead of a binary heap. The way the Leonardo heap is represented in-place in the array (or list) enables sorting operations to take advantage of partially sorted data (data where there are runs of sorted elements). This gives the algorithm a linear best case.
