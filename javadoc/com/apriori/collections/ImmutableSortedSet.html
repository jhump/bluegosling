Interface ImmutableSortedSet<E>

  • Type Parameters:
    E - the type of element in the set
    All Superinterfaces:
    ImmutableCollection<E>, ImmutableSet<E>, Iterable<E>

    public interface ImmutableSortedSet<E>
extends ImmutableSet<E>
    An immutable, read-only set that maintains a total ordering over its elements. This is analogous to a NavigableSet except that it provides no mutation operations.
