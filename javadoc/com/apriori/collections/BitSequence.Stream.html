Interface BitSequence.Stream

  • Enclosing interface:
    BitSequence

    public static interface BitSequence.Stream
    A stream of bits. This interface is similar to an iterator, except it allows for selecting an arbitrary number of bits with each fetch.

      • next

        long next(int numberOfBits,
        BitSequence.BitOrder order)
        Returns the next chunk of bits in the stream. Since the bits are returned as a primitive long, the chunk size must be 64 bits or less. The next bit in the sequence will either be the least or most significant bit in the returned value. For example, if the next six bits in the sequence are as follows: 
        0 1 0 0 1 1
        Then fetching the next six bits in LSB order will return the number 50, which is 110010 in binary. In MSB order, the fetch will return the number 19, or 010011 in binary.
        Parameters:
        numberOfBits - the number of bits to fetch
        order - the order of bits in the returned value
        Returns:
        the next chunk of bits
        Throws:
        NoSuchElementException - if the requested number of bits is greater than the number of bits remaining in the stream
