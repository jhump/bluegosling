Interface RandomAccessNavigableMap<K,V>

  • Type Parameters:
    K - the type of keys in the map
    V - the type of values in the map
    All Superinterfaces:

    All Known Implementing Classes:
    AbstractRandomAccessNavigableMap, WeightBalancedTreeMap

    public interface RandomAccessNavigableMap<K,V>
extends NavigableMap<K,V>
    A navigable map that provides random access over its elements. The entries in the map are ordered according to the map's Comparator or (if there is none) according to the keys' natural ordering.
