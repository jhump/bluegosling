com.apriori.apt.testing

Annotation Type InitializeProcessorField


  • @Target(value=TYPE)
@Retention(value=RUNTIME)
@Inherited
public @interface InitializeProcessorField
    Indicates the name of a field on the test class that holds a Processor that should be initialized with the current ProcessingEnvironment prior to running each test case.

    This annotation should not be used in conjunction with ProcessorUnderTest. Use one or the other. This annotation is useful if you need to construct or initialize the processor programmatically in a @Before method.
