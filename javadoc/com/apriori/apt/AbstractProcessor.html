    public abstract class AbstractProcessor
extends AbstractProcessor
    An abstract annotation processor that makes for a better base class than the standard. This class provides the following two new features:
    • This processor sets up thread-local state about the current processing environment. This makes it easier for APIs to work with elements and type mirrors without requiring that callers supply the environment. See ProcessingEnvironments for more information.
    • The set of supported annotation types can be defined by referencing actual annotation class tokens using a @SupportedAnnotationClasses annotation on the processor. Note that this does not provide any wildcard facilities, but can be used in conjunction with the standard @SupportedAnnotationTypes annotation, which doess.
