Interface Type

  • All Known Implementing Classes:
    Class, GenericArrayType, ParameterizedType, TypeVariable, WildcardType

    public interface Type
    A interface that represents a generic type. Using implementors of this interface, information on generic type variables and even reifiable type parameters can be programmatically determined for processed type elements.

    This mirrors the java.lang.reflect.Type interface, but is for representing types in source form for annotation processing vs. types in compiled form at runtime.
