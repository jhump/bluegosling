Interface AnnotatedElement

  • All Known Implementing Classes:
    Class, Constructor, Field, Method, Package, Parameter

    public interface AnnotatedElement
    An element that can have annotations. This includes representations of things in the Java programming language which can be annotated, like packages, types, fields, constructors, methods, and parameters.

    The Java programming language also allows local variables (including exception variables defined in catch blocks) to be annotated, but support for this during annotation processing is incomplete, so there currently are no such implementations.

    This interface is analogous to java.lang.reflect.AnnotatedElement, except that it represents a source-level type (during annotation processing) instead of an actual runtime type. Unlike java.lang.reflect.AnnotatedElement, this interface can be used to inspect annotations whose retention policy is RetentionPolicy.CLASS or RetentionPolicy.SOURCE.

      • FROM_ELEMENT

        static final Function<Element,AnnotatedElement> FROM_ELEMENT
        A function that transforms a Element to an AnnotatedElement.

        The function will return instances of Class, Field, Method, Constructor, Parameter, or Package. If the specified input element does not represent one of these types of elements then null is returned.
