com.apriori.apt.reflect

Class Parameter<M extends ExecutableMember>

  • java.lang.Object
    • com.apriori.apt.reflect.Parameter<M>
  • Type Parameters:
    M - the type of executable member (constructor or method) to which the parameter belongs
    All Implemented Interfaces:
    AnnotatedElement

    public class Parameter<M extends ExecutableMember>
extends Object
    A representation of a parameter to an executable member, like a constructor or method. Unlike many of the other types in this package, this one has no analog in the java.lang.reflect package since parameter details aren't stored in class files and thus aren't reified for the benefit of runtime reflection. The java.lang.reflect APIs do have ways to get at parameter details (via java.lang.reflect.Method and java.lang.reflect.Constructor classes), but they are lacking one key attribute of a parameter that is available in Java source but not in runtime types: the parameter name.
