com.apriori.apt.reflect

Class ArRoundEnvironment

  • java.lang.Object
    • com.apriori.apt.reflect.ArRoundEnvironment

  • public class ArRoundEnvironment
extends Object
    The environment for a single round of annotation processing. This differs from the standard RoundEnvironment in that its API is in terms of the reflection API in this package instead of in terms of elements, annotation mirrors, and type mirrors.
    See Also:
    RoundEnvironment, ArProcessor
