Interface ArType.Visitor<R,P>

  • Type Parameters:
    R - the type of value returned by the visitor
    P - an optional parameter that is passed to the visitor (if not needed, use Void)
    Enclosing interface:
    ArType

    public static interface ArType.Visitor<R,P>
    A visitor for simulating multiple dispatch over the concrete implementations of ArType. This can be used to more elegantly handle polymorphism of types than using numerous if-else type checks (either using instanceof or looking at the value returned from ArType.getTypeKind()) and then downcasts (or equivalent calls to methods in ArTypes, such as ArTypes.asClass(ArType)).
    See Also:
