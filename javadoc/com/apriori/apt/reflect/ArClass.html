Class ArClass

  • java.lang.Object
    • com.apriori.apt.reflect.ArClass
  • All Implemented Interfaces:
    ArAnnotatedElement, ArGenericDeclaration, ArType

    public abstract class ArClass
extends Object
implements ArAnnotatedElement, ArGenericDeclaration, ArType
    A class (or interface, enum, or annotation type). This is analogous to Class except that it represents a type in source form vs. in compiled form at runtime.

    Sadly, since the types here may not be available as actual Class tokens, this type cannot be parameterized the way that Class is.

        If this is not a top-level class (i.e. it is an anonymous, local, or member class) then this will be the fully-qualified name of the enclosing class, followed by a dollar sign, and then followed by this class's simple name (or a generated name if this is an anonymous class).

        Reference types are encoded with an "L" followed by the type's fully-qualified name and then a terminating ";".
