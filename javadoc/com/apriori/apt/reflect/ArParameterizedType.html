Class ArParameterizedType

  • java.lang.Object
    • com.apriori.apt.reflect.ArParameterizedType
  • All Implemented Interfaces:
    ArType

    public class ArParameterizedType
extends Object
implements ArType
    An instance of type with information about actual type parameters. Even though a ArClass has type variables, it represents a raw or erased type and its type variables represent type variable declarations. A parameterized type represents an actual instance -- or usage -- of a type with actual type parameter values. This class is analogous to java.lang.reflect.ParameterizedType, except that it represents types in Java source (during annotation processing) vs. representing runtime types.

      • getOwnerType

        public ArType getOwnerType()
        Returns the enclosing type. If this is a top-level type then this method returns null. The enclosing type could also be a ArParameterizedType, or it could be a ArClass.

        For example, if this type is ParentType<T>.EnclosedType<S> then this function would return a ArParameterizedType that represents ParentType<T>. If this is a static member type or a member of a type that is not parameterized, then it might look more like the type ParentType.EnclosedStaticType<S>, in which case this method will return a ArClass that represents ParentType.
