    All Known Implementing Classes:

    public interface Member
    A member of a Java class or interface. Members include fields (including enum constants), methods (including annotation methods), and constructors. This is analogous to java.lang.reflect.Member, except that it represents members in Java source (during annotation processing) vs. representing members of runtime types or elements (methods and constructors) of runtime types.
