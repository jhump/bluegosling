Class Annotation

  • java.lang.Object
    • com.apriori.apt.reflect.Annotation

  • public class Annotation
extends Object
    An annotation in Java source code. It is sort of analogous to the actual java.lang.annotation.Annotation interface, except that it represents source-level annotations during annotation processing vs. annotations on actual runtime types.

      • getAnnotationAttributes

        public Map<String,?> getAnnotationAttributes()
        Returns the annotation's attributes and values, including undeclared attributes with their corresponding default value. The keys represent the annotation's methods; the values represent the actual values. The values will be instances of one of the following:
        • Strings and boxed primitives.
        • Class objects for methods whose return is a java.lang.Class.
        • List<?> for methods whose return type is an array. The values in this list will be one the other types described here.
        • Other Annotations for methods that return other annotation types.
        Returns:
        the annotation's attributes and values
        See Also:
        getDeclaredAnnotationAttributes()

        Returns the annotation's attributes and values. Only values explicitly declared in the annotation are included. The keys represent the annotation's methods; the values represent the actual values. See getAnnotationAttributes() for more details.
