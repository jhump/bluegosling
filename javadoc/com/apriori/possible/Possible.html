Interface Possible<T>

  • Type Parameters:
    T - the type of the possible value
    All Known Subinterfaces:
    Fulfillable<T>
    All Known Implementing Classes:
    Holder, Optional, Reference

    public interface Possible<T>
    A possible value. A value might be present, but it may not be. Some implementations may allow a "present" value to be null. Others may treat null the same as an absent value. Some implementations may be mutable and provide additional operations for setting or clearing the value.
