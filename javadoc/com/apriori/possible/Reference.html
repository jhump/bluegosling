com.apriori.possible

Class Reference<T>

  • java.lang.Object
    • com.apriori.possible.Reference<T>
  • Type Parameters:
    T - the type of the possible value
    All Implemented Interfaces:
    Possible<T>

    public abstract class Reference<T>
extends Object
implements Possible<T>
    A possible value that, even if present, can be null. A reference can either be set or unset. If set, a value is present; if unset, the value is absent. References are immutable. A similar possible value that is mutable is Holder.

      • cast

        public static <S,T> Reference<T> cast(Reference<S> from,
                      Class<T> target)
        Casts a reference to a different type. The present value must be assignable to the new type. Using this method is much more efficient than creating a new reference value with a different type. Since reference values are immutable, the type on an existing instance can safely be re-cast when a present value is null or an instance of the cast type.
        Parameters:
        from - the reference value from which we are casting
        target - the type to which the value will be cast
        Returns:
        this but re-cast to a Reference<T> where T is the target type
        Throws:
        ClassCastException - if the reference is set but the value is neither an instance of the specified type nor null
