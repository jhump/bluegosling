Class Blaster.Result<T>

  • java.lang.Object
    • com.apriori.reflect.Blaster.Result<T>

    T - the type of the result
    Enclosing class:
    Blaster<T>

    public static class Blaster.Result<T>
extends Object
    The result of blasting a method invocation out to a single target object. This result can either be a returned value or a thrown exception, depending on what happened during the method invocation.
