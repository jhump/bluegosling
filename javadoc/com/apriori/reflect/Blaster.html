com.apriori.reflect

Class Blaster<T>

  • java.lang.Object
    • com.apriori.reflect.Blaster<T>
  • Type Parameters:
    T - the interface type whose methods are "blasted"

    public class Blaster<T>
extends Object
    Forwards method calls to multiple instances. Effectively, this broadcasts methods invoked on a "Blaster" (proxy) out to multiple objects that implement the same interface. In other words, it "blasts" method calls out to multiple objects.

    If a method is invoked that returns an interface, then the proxy returns another blaster, that sends method invocations to all such interfaces returned by underlying objects. If a method is invoked that does not return an interface, then the proxy returns the last successful call to one of the target objects or a default value for the return type if none of the method invocations were successful.

    Exceptions thrown by the target objects are suppressed, but available by inspecting the full set of results after a method call. Similarly, for methods that do not return an interface, if you want to know all results (instead of only the last successful one), you can inspect the full set of results.

      • blasted

        public static <T> List<T> blasted(T t)
        Gets the list of values returned from the most recent method invocation on a blasting proxy on this thread. This method is only useful if all blasted method invocations succeeded. Otherwise, this will throw an exception.

        The argument is for readability, so code can do the following:

         SomeInterface o1, o2, o3;
 SomeInterface blaster = Blaster.blastTo(SomeInterface.class, o1, o2, o3);
 // Just calling blaster.getIntegerValue() would only return the last
 // successful return value, but we want all of them.
 List<Integer> ints = blasted(blaster.getIntegerValue());

        If no invocation has occurred on this thread or the last such invocation was aborted then this will throw an exception.

        Parameters:
        t - a dummy argument, for readability using the idiom described above
        Returns:
        the list of values returned from the last blasted method invocation
        Throws:
        IllegalStateException - if no invocation has occurred on this thread or the last such invocation was aborted
        Error - if any of the blaster's target instances threw an Error during method invocation
        RuntimeException - if any of the blaster's target instances threw an exception during method invocation
        See Also:
        blastedResults()
