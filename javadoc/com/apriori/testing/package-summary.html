Package com.apriori.testing Description

Tools for testing implementations of interfaces. The main class here is for verifying that one implementation of an interface conforms to specs by comparing its behavior against a "known good" implementation of that interface. Other useful classes can also be found, but they are generally in support of the main InterfaceVerifier.
